# ドット絵のサイズをいい感じに拡大縮小するやつ

するやつ。

## URL

https://irokaru.github.io/pixel-scaler/

## 各種パッケージインスコ
```
yarn install
```

### ホットリロードしてくれる開発モード
```
yarn serve
```

```
yarn electron:serve
```

### プロダクション向けコンパイル
```
yarn build
```

### lint
```
yarn lint
```

### デプロイ
```
yarn deploy
```

### デスクトップアプリケーション作成
```
yarn electron:build
```
